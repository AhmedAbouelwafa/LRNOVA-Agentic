# LRNOVA

AI platform for generating educational content — videos, courses, and slides.

## Quick Start

```bash
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200)

## Stack

Angular 21 · TypeScript 5.9 · RxJS 7 · Vitest

## Scripts

| Command | Description |
|---|---|
| `npm start` | Dev server on `localhost:4200` |
| `npm run build` | Production build to `dist/` |
| `npm run watch` | Watch mode (dev config) |
| `npm test` | Run unit tests |

## Agents

- **Video** — Generate AI video lessons
- **Text** — Create courses, scripts, and assessments
- **Slides** — Build interactive presentations

## Key Routes

| Path | Page |
|---|---|
| `/` | Home — prompt input and agent selection |
| `/results` | Results — generated content and chat |
| `/pricing` | Pricing plans |

## Languages

English and Arabic supported. Language and RTL direction switch at runtime.
